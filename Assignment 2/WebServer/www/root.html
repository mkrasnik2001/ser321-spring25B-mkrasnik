You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html

Newly added endpoints:

  • /weather -- returns the temperature of the location you gave with the coordinates. The format should be: query={latitude},{longitude} | example: /weather?query=37,-145
  • /bmi - returns your BMI index given your weight in kilograms and height in meters. The format should be: height={height}&weight={weight} | example: /bmi?height=1.9&weight=85
